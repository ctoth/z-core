# z-core

`z-core` is a from-scratch Rust Z180 CPU/SoC emulator core for native, Python, and WebAssembly consumers. It is being built to give the `qns` Braille 'N Speak emulator an owned, portable replacement for its current GPLv2 CPU core, including a future path to iOS. The project is currently in its initial scaffold phase and does not yet implement CPU behavior.
